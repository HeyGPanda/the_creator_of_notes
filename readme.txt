                                            Документация к проекту

Название проекта: Менеджер заметок
Описание: Менеджер заметок - это программа, которая позволяет пользователям создавать, просматривать, искать и удалять заметки. Программа использует базу данных SQLite для хранения заметок.


                                                Функции программы

Добавление заметки: Пользователь может ввести заголовок и содержание новой заметки, после чего заметка будет сохранена в базе данных.
Просмотр списка заметок: Пользователь может просмотреть список всех сохраненных заметок с их заголовками и содержанием.
Поиск заметок: Пользователь может выполнить поиск заметок по ключевому слову. Результаты поиска будут отображены с соответствующими заголовками и содержанием.
Удаление заметки: Пользователь может удалить заметку из базы данных, указав ее содержание.
Инструкция по запуску и тестированию программы:

Установите Python версии 3.x на ваш компьютер, если он ещё не установлен.
Скачайте файл с исходным кодом программы и сохраните его на вашем компьютере.
Установите модуль SQLite3, если он ещё не установлен, с помощью команды pip install pysqlite3.
Откройте командную строку или терминал и перейдите в каталог, содержащий файл с исходным кодом программы.
Запустите программу, выполнив команду python <название_файла>.py, где <название_файла> - имя файла с исходным кодом программы.
Взаимодействуйте с программой, следуя инструкциям в меню.
